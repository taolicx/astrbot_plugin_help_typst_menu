# 文字菜单助手

面向群聊用户的 AstrBot 纯文字菜单插件。

这个版本不再生成图片，也不依赖 Typst。菜单会直接以文字消息发送，适合想要轻量、稳定、方便复制的插件指令索引。

## 功能

- `/helps`：显示一级菜单，只展示插件中文名称、序号和指令数量。
- `/helps 序号`：查看某个插件的详细指令，例如 `/helps 1`。
- `/helps 插件名`：按中文名、插件 ID 或描述搜索插件，例如 `/helps 每日签到`。
- `/events 关键词`：查看事件监听列表。
- `/filters 关键词`：查看过滤器列表。

一级菜单会优先显示插件配置里的中文展示名；没有中文展示名时，会使用内置中文别名或从中文描述中生成短名称，避免群友看到一堆英文插件 ID。

## 依赖

```text
pydantic
```

## 配置

- `enable_waiting_message`：查询菜单时是否先发送等待提示。
- `ignored_plugins`：不希望展示在菜单里的插件 ID 列表。

## 使用示例

```text
/helps
/helps 3
/helps 每日签到
```

## 安装目录名

建议安装为：

```text
astrbot_plugin_text_menu
```
